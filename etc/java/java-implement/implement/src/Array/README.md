# 배열의 활용

---

## Basic

1. 총합과 평균
> 배열의 모든 요소를 더해서 총합과 평균을 구한다.

2. 최대값과 최소값
> 배열의 요소 중에서 제일 큰 값과 제일 작은 값을 찾는다.

3. 섞기 (Shuffle)
> 배열의 요소의 순서를 반복해서 바꾼다. (카드섞기, 로또번호 생성)

4. 임의의 값으로 배열 채우기
> 연속 또는 불연속적인 값들로 배열을 초기화 한다.

5. 정렬하기 (sort)
> 오름차순, 내림차순으로 배열을 정렬

6. 빈도수 구하기
> 배열에 어떤 값이 몇 개 저장되어 있는지 세어서 보여준다.
